## Locked decisions (2026-07-05)

1. **Mesh** — a procedural subdivided plane sized to the domain extent, generated by us, density from a
   tier value. Built behind a thin mesh-strategy seam so a Phase-2 camera-centred clipmap/LOD grid
   (the Crest/KWS approach for large/infinite water) can replace it without touching the surface shader.
   Bounded pond = the degenerate single-density case, mirroring "bounded = depth-1 cascade."
2. **Render path** — start with a transparent `MeshRenderer` whose material reads the field textures +
   extent as **global** shader properties, published each frame by the feature after compose. Add
   reflection as its own pass. Migrate the surface draw to a custom `SurfacePass` (doc §5) only if URP's
   transparent queue proves limiting. Refraction uses URP's `_CameraOpaqueTexture` (enable Opaque
   Texture — and Depth Texture — on the URP asset).
3. **Look** — refraction + reflection is the target, built incrementally (see build order), not all at once.
4. **Shader** — hand-written HLSL URP shader in the package (code-first, DRY with our constants). No Shader Graph.

## Build order (commit each green step; discuss + get a yes before writing)

1. **Field globals publisher.** After compose, set global textures (`_UnifiedWater_Dynamic`,
   `_UnifiedWater_NormalFoam`) and extent uniforms (`_UnifiedWater_Center`, `_UnifiedWater_Size`,
   `_UnifiedWater_TexelSize`) from the primary registered domain. One publisher; the HLSL mapping mirrors
   `BoundedDomainExtent` (decision #4's mirror). Names live in a constants class, not inline strings.
2. **Procedural surface mesh + authoring.** Generate a subdivided plane sized to the extent (density from a
   new tier field, e.g. `SurfaceMeshResolution`, decoupled from field texture resolution). Behind an
   `ISurfaceMesh`-style seam. A `MeshFilter` + `MeshRenderer` carry the water material.
3. **`WaterSurface.shader` (HLSL, URP, transparent).** Vertex displaces Y by sampling
   `_UnifiedWater_Dynamic` height in world→uv (× a `heightScale` authoring/tier value); fragment uses the
   `_UnifiedWater_NormalFoam` normal + a water tint + Fresnel + specular. → visible 3D lit ripples.
4. **Refraction.** Sample `_CameraOpaqueTexture` with normal-based UV distortion; depth-based absorption tint.
5. **Reflection.** Planar reflection pass or SSR — decide the mode at this step (`ReflectionMode` enum already
   drafted in the architecture doc).
